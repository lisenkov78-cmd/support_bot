ГОТОВО ДЛЯ TIMEWEB CLOUD → APP PLATFORM → DOCKER COMPOSE (БЕЗ ПРАВОК В КОДЕ)

Что делаем:
1) Создай репозиторий на GitHub.
2) Загрузите в КОРЕНЬ репозитория файлы:
   - support_bot.py (как есть)
   - requirements.txt
   - Dockerfile
   - docker-compose.yml
3) Timeweb Cloud → App Platform → Создать приложение → Docker Compose → выбрать репозиторий.
4) Нажми Deploy / Redeploy.
5) Проверяй Logs.

Важно:
- Токен берётся из самого support_bot.py. В Variables ничего добавлять не нужно.
